# Raspberry-Pi-TFT-Web-Image-Controller-ST7789-SPI-Display-
Web-based controller for a 2.4" ST7789 SPI TFT display using Raspberry Pi. Upload, crop, zoom, and invert images through a browser interface and display them in real time on the TFT screen via SPI. Built with Python, Flask, and Pillow, and compatible with Raspberry Pi OS Lite.
